# RL Snake

<img src="images/RLSnake.png">

Simple Snake Game in C++ with SFML
## How to compile this project
Clone this repository
```bash
$ git clone https://github.com/Renan2010/RLSnake.git
```
Enter the directory
```bash
$ cd RLSnake
```
And run ninja to compile this game
```bash
$ ninja -j$(nproc)
```
